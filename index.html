Política de Privacidade - JS Arena

A sua privacidade é importante para nós. Esta política explica como o JS Arena e o desenvolvedor Js Music Brazil / CapybaraClickerJS lidam com informações.

1. Coleta de Dados e Firebase

O JS Arena utiliza o Google Firebase Realtime Database para permitir partidas online. Ao jogar no modo online, os dados das jogadas (posição no tabuleiro e status da partida) são transmitidos temporariamente para os servidores do Firebase para sincronização entre os jogadores.

2. Informações Pessoais

Não coletamos informações pessoais identificáveis como nome real, e-mail, CPF ou geolocalização. O ID da sala criado pelo usuário é usado apenas para conectar dois jogadores na mesma partida.

3. Armazenamento Local

Podemos utilizar tecnologias de armazenamento local (Local Storage) no seu navegador para salvar preferências simples, como o último modo de jogo selecionado ou recordes locais.

4. Links de Terceiros

Nosso portal pode conter links para nossas redes sociais oficiais (YouTube Js Music Brazil, TikTok e Kwai). Ao clicar nesses links, você estará sujeito às políticas de privacidade dessas plataformas externas.

5. Hospedagem

Este site é hospedado na plataforma Vercel, que pode coletar logs técnicos básicos (como endereço IP e tipo de navegador) para garantir a segurança e o funcionamento do serviço, conforme as normas da plataforma.
